# ddtdanilo.github.io

Personal portfolio website for Danilo Díaz Tarascó — CTO & Electronic Engineer.

**Live:** [ddtdanilo.github.io](https://ddtdanilo.github.io)

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6)
- [Three.js](https://threejs.org/) — 3D particle background
- [GSAP](https://greensock.com/gsap/) + ScrollTrigger — Animations
- Google Fonts (Space Grotesk, JetBrains Mono)

## Features

- Bilingual (EN/ES) with auto-detection and toggle
- Immersive 3D particle field with mouse interaction
- Responsive design (mobile-first)
- Custom cursor effects
- Glassmorphism UI
- SEO optimized (JSON-LD, Open Graph, sitemap)
- Consulting booking page

## Deployment

GitHub Pages from `master` branch. Push to deploy.

## Structure

```
index.html          # Main portfolio
consult.html        # Consulting services page
consult-success.html # Booking confirmation
assets/css/style.css # Styles
assets/js/main.js    # JS (i18n, Three.js, GSAP, interactions)
assets/img/          # Images and portfolio screenshots
llms.txt            # LLM-friendly plain text profile
```
